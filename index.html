printf() Function

The printf() function is used to print formatted output to the standard output (usually the console).

Syntax

int printf(const char *format, ...);

Format Specifiers

Specifier Description
%d or %i Signed decimal integer
%u Unsigned decimal integer
%f Decimal floating point
%c Character
%s String
%p Pointer address

Examples

// Basic printf examples
printf("Hello, World!\n");
printf("Integer: %d, Float: %.2f, Char: %c\n", 42, 3.14159, 'A');

Data Types

C has several built-in data types that can be used to declare variables.

Basic Data Types

Type Size (bytes) Description
char 1 Character or small integer
int 4 Integer
float 4 Single-precision floating point
double 8 Double-precision floating point
void - No type

Type Modifiers

Type modifiers can alter the meaning of basic data types:

  • signed - Default for char and int (numbers can be positive or negative)
  • unsigned - Only positive numbers
  • short - Smaller size than int
  • long - Larger size than int

Examples

// Variable declarations
char letter = 'A';
int age = 25;
float temperature = 98.6;
double pi = 3.1415926535;
unsigned int positiveNumber = 100;
long long bigNumber = 123456789012345;

Arithmetic Operations

C supports various arithmetic operations for performing calculations.

Basic Arithmetic Operators

Operator Description Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
% Modulus (remainder) a % b

Compound Assignment Operators

Operator Equivalent to Example
+= a = a + b a += b
-= a = a - b a -= b
*= a = a * b a *= b
/= a = a / b a /= b
%= a = a % b a %= b

Examples

int a = 10, b = 3;
int sum = a + b;      // 13
int difference = a - b; // 7
int product = a * b;   // 30
float quotient = (float)a / b; // 3.333...
int remainder = a % b; // 1

a += 5;  // a is now 15
b *= 2;  // b is now 6

Loops

Loops are used to execute a block of code repeatedly until a specified condition is met.

Types of Loops

Loop Type Description
while Repeats a block while condition is true
do-while Executes once, then repeats while condition is true
for Repeats a block for a specific number of times

Examples

// while loop
int i = 0;
while (i < 5) {
    printf("%d ", i);
    i++;
} // Output: 0 1 2 3 4

// do-while loop
int j = 0;
do {
    printf("%d ", j);
    j++;
} while (j < 5); // Output: 0 1 2 3 4

// for loop
for (int k = 0; k < 5; k++) {
    printf("%d ", k);
} // Output: 0 1 2 3 4

Loop Control Statements

Statement Description
break Terminates the loop immediately
continue Skips the current iteration and continues with the next

Branching

Branching statements allow your program to make decisions and execute different code blocks based on conditions.

if Statement

if (condition) {
    // code to execute if condition is true
}

if-else Statement

if (condition) {
    // code to execute if condition is true
} else {
    // code to execute if condition is false
}

else-if Ladder

if (condition1) {
    // code if condition1 is true
} else if (condition2) {
    // code if condition2 is true
} else {
    // code if all conditions are false
}

switch Statement

switch (expression) {
    case value1:
        // code for value1
        break;
    case value2:
        // code for value2
        break;
    default:
        // code if no cases match
}

Ternary Operator

result = condition ? value_if_true : value_if_false;

Examples

// if-else example
int num = 10;
if (num > 0) {
    printf("Positive number\n");
} else if (num < 0) {
    printf("Negative number\n");
} else {
    printf("Zero\n");
}

// switch example
char grade = 'B';
switch (grade) {
    case 'A':
        printf("Excellent!\n");
        break;
    case 'B':
        printf("Well done\n");
        break;
    default:
        printf("Invalid grade\n");
}

// ternary operator example
int a = 5, b = 10;
int max = (a > b) ? a : b;

Structures

Structures (structs) allow you to create custom data types that group together variables of different types.

Defining a Structure

struct structureName {
    dataType member1;
    dataType member2;
    // ... more members
};

Declaring Structure Variables

// Method 1: Declare when defining
struct Person {
    char name[50];
    int age;
    float height;
} person1, person2;

// Method 2: Declare separately
struct Person person3;

Accessing Structure Members

// Using dot operator
struct Person p1;
strcpy(p1.name, "John");
p1.age = 30;
p1.height = 5.9;

// Using pointer and arrow operator
struct Person *ptr = &p1;
printf("Name: %s\n", ptr->name);

typedef with Structures

// Creates an alias for the structure
typedef struct {
    char make[20];
    int year;
} Car;

// Now we can declare variables without 'struct' keyword
Car myCar;

Nested Structures

struct Address {
    char street[50];
    char city[30];
};

struct Employee {
    char name[50];
    int id;
    struct Address addr; // Nested structure
};

struct Employee emp1;
strcpy(emp1.addr.city, "New York");

Arrays

Arrays are used to store multiple values of the same type in a single variable.

Declaring Arrays

dataType arrayName[arraySize];

Initializing Arrays

// Method 1: Initialize at declaration
int numbers[5] = {1, 2, 3, 4, 5};

// Method 2: Initialize without size
float temperatures[] = {98.6, 99.2, 97.8};

// Method 3: Initialize element by element
char vowels[5];
vowels[0] = 'a';
vowels[1] = 'e';
// ...

Accessing Array Elements

int numbers[3] = {10, 20, 30};
int first = numbers[0]; // 10
numbers[1] = 25;       // Change second element to 25

Multidimensional Arrays

// 2D array (matrix)
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

// Accessing elements
int val = matrix[1][2]; // 6

Arrays and Functions

// Passing array to function
void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    printArray(numbers, 5);
    return 0;
}

#define Directive

The #define directive is used to create macros, which are fragments of code that are replaced by the preprocessor before compilation.

Simple Macros

#define PI 3.14159
#define MAX_SIZE 100
#define GREETING "Hello, World!"

Function-like Macros

#define SQUARE(x) ((x) * (x))
#define MAX(a, b) (((a) > (b)) ? (a) : (b))

// Usage:
int result = SQUARE(5); // 25
int bigger = MAX(10, 20); // 20

Multi-line Macros

#define PRINT_ERROR(msg) \
    printf("Error: %s\n", msg); \
    printf("File: %s, Line: %d\n", __FILE__, __LINE__)

Predefined Macros

Macro Description
__DATE__ Current date as a string
__TIME__ Current time as a string
__FILE__ Current filename as a string
__LINE__ Current line number as an integer

Conditional Compilation

#define DEBUG 1

#if DEBUG
    printf("Debug mode is on\n");
#else
    printf("Debug mode is off\n");
#endif

Standard Libraries

C provides standard libraries that contain functions for common operations. These libraries are included using the #include directive.

Common Standard Libraries

Library Description Example Functions
stdio.h Standard Input/Output printf, scanf, fopen, fclose
stdlib.h Standard Library malloc, free, exit, rand
string.h String Operations strcpy, strlen, strcmp
math.h Mathematical Functions sqrt, pow, sin, cos
time.h Time and Date time, clock, difftime

Including Libraries

#include <stdio.h>  // Standard I/O functions
#include <math.h>   // Math functions
#include "myheader.h" // User-defined header

Example Usage

#include <stdio.h>
#include <string.h>
#include <math.h>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    
    strcat(str1, str2); // Concatenate strings
    printf("%s\n", str1); // "HelloWorld"
    
    double root = sqrt(25.0); // Square root
    printf("Square root: %.2f\n", root); // 5.00
    
    return 0;
}

File Input/Output

C provides functions for reading from and writing to files. File operations are performed using the FILE pointer.

Opening a File

FILE *filePointer;
filePointer = fopen("filename.txt", "mode");

File Modes

Mode Description
"r" Open for reading (file must exist)
"w" Open for writing (creates new or truncates existing)
"a" Open for appending (creates new if doesn't exist)
"r+" Open for reading and writing (file must exist)
"w+" Open for reading and writing (creates new or truncates)
"a+" Open for reading and appending

Closing a File

fclose(filePointer);

Writing to a File

// Write formatted data
fprintf(filePointer, "Formatted data: %d %f %s\n", intVar, floatVar, stringVar);

// Write a character
fputc('A', filePointer);

// Write a string
fputs("Hello, File!\n", filePointer);

Reading from a File

// Read formatted data
fscanf(filePointer, "%d %f %s", &intVar, &floatVar, stringVar);

// Read a character
char ch = fgetc(filePointer);

// Read a string (until newline or EOF)
fgets(buffer, size, filePointer);

Complete Example

#include <stdio.h>

int main() {
    // Writing to a file
    FILE *fptr = fopen("example.txt", "w");
    if (fptr == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    fprintf(fptr, "This is a test file.\n");
    fprintf(fptr, "Writing numbers: %d %f\n", 42, 3.14159);
    fclose(fptr);
    
    // Reading from a file
    fptr = fopen("example.txt", "r");
    char buffer[100];
    
    while (fgets(buffer, 100, fptr) != NULL) {
        printf("%s", buffer);
    }
    
    fclose(fptr);
    return 0;
}
